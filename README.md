# Java-Project

## Overview  
This repository contains a collection of simple Java-based tools and mini-projects, bundled together to demonstrate a variety of functionalities and programming concepts.

## Tools / Projects Included  
Here are some of the tools and mini-projects you’ll find in this collection:

- To-Do List  
- Temperature Converter (°C ↔ °F ↔ K)  
- Password Generator  
- Basic Calculator with History  
- File Word Counter (words, characters, lines)  
- URL Slug Generator  
- Random Quote Generator  
- Palindrome Checker  
- Prime Number Checker  
- Interest Calculator (Simple & Compound)  
- BMI Calculator  
- Live Clock (Date & Time Display)  
- Sudoku Game (9×9 puzzle)  
- Simple Chatbot  
- Tic-Tac-Toe (2-Player Game)  

## Getting Started  
### Prerequisites  
- Java JDK (version 8 or higher)  
- A code editor or IDE such as IntelliJ IDEA, Eclipse, or VS Code
